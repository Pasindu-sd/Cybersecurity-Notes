What is Bug Bounty?
A Bug Bounty is a program where companies pay ethical hackers to find and report security vulnerabilities in their websites, apps, APIs, or systems.


You legally test their systems (within their allowed scope) and if you find a bug, they reward you with:

   - Money
   - Hall of Fame recognition
   - Swag / gifts
   - Reputation points

### What Bug Bounty Hunters Do

   - Find weaknesses in websites, mobile apps, APIs, servers.
   - Test for vulnerabilities like SQL Injection, XSS, IDOR, CSRF, etc.
   - Create a Proof of Concept (PoC) showing how the bug works.
   - Report it responsibly to the company.
   - Get rewarded if the bug is valid.

### Popular Bug Bounty Platforms

   - HackerOne
   - Bugcrowd
   - Intigriti
   - YesWeHack
   - OpenBugBounty

### Why Bug Bounty is Good?

   - You learn real cybersecurity skills
   - You can earn money
   - You build a strong reputation
   - Helps you get internships or jobs in cybersecurity
